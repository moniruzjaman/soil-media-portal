⚙️ Google Drive Setup

Stored in browser localStorage only — never sent anywhere except Google APIs.

1

Go to Google Cloud Console → Enable Google Drive API.

2

APIs & Services → Credentials → Create API Key. Restrict it to Google Drive API only.

3

In Google Drive → right-click folder → Share → Anyone with the link → Viewer. Folder ID is the segment after /folders/ in the URL.

4

Paste the key above → Save → click Load ▶ in the header.
