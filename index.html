Professional Summary

Abu Md. Moniruzzaman is a visionary and dedicated officer of the Department of Agricultural Extension (DAE), Government of the People's Republic of Bangladesh, serving as Additional Deputy Director (Horticulture) at Kurigram. A member of the 31st BCS (Agriculture) cadre, he brings a distinctive blend of corporate leadership, entrepreneurial acumen, and field-level extension expertise to Bangladesh's agricultural sector.

With over 8 years of field-level experience across geographically diverse settings including haor wetlands, hill tracts, and plains, he has consistently championed sustainable agriculture, digital extension services, and evidence-based crop management for smallholder farmers across Bangladesh.

Digital Innovation & AgriTech Leadership

Moniruzzaman is a recognized Digital Transformer in Bangladesh's agricultural extension ecosystem. He actively develops and deploys AI-powered, data-driven tools that bring modern agricultural advisory services directly to farmers' doorsteps:

🌿 CABI Diagnosis Web App (2025–Present)

A pioneering React/Vite web application integrating Claude AI for crop disease diagnosis based on the globally acclaimed CABI Plantwise Ready Reckoner methodology. Features AI-powered diagnostic protocol, bilingual Bengali/English output with voice input, 94+ DAE-registered pesticides dataset, real-time weather integration, and Plantix-inspired UI.

📱 Krishi AI Mobile App

A React Native mobile application with Cloudflare Worker backend for AI-based crop advisory. Features MobileNetV2 TFLite crop disease detection, 17 agricultural tool screens with Bengali error messages, and zero API keys in the APK for secure deployment via EAS build pipeline.

🪪 কৃষক কার্ড — Krishak Card App

A Vercel-deployed React application serving as a digital farmer services portal. Includes live DAE market prices via market.dam.gov.bd with district selector, 30-minute edge caching, 11 integrated farmer services, and Noto Serif Bengali branding with Bangladesh national emblem.

🎮 স্বশিক্ষা কৃষি প্রকল্প — IPM/FFS Educational Game

A single-file HTML5 Bengali-language IPM decision game teaching a five-step methodology across three rice scenarios (Leaf Blast, Stem Borer, Nitrogen Deficiency), incorporating AI-generated SVG crop illustrations via the Anthropic API.

📺 AgriWisdom YouTube Channel (@AgriWisdomBd)

A Bengali-language agricultural extension YouTube channel producing AI-generated explainer videos on modern farming techniques, digital tools, and policy matters, semi-automated via Google Sheets + Apps Script with Claude API.

⚙️ Setup Guide

Key is stored only in your browser's local storage — never sent anywhere except Google APIs.

1

Go to Google Cloud Console → Create a project → Enable Google Drive API.

2

Go to APIs & Services → Credentials → Create Credentials → API Key. Copy the key and paste it above.

3

Restrict the key: under API restrictions → select Google Drive API. Under Application restrictions → select HTTP referrers and add your site URL.

4

Open your Google Drive folder → click Share → change access to "Anyone with the link". The folder ID is the part of the URL after /folders/.

5

Paste the folder ID or full URL into the header bar and click Load ▶.
